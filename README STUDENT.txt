Student Dropout Prediction with Machine Learning

Este projeto aplica técnicas de machine learning para prever a evasão de alunos em uma instituição de ensino superior, com foco em intervenção precoce baseada nos dados do 1º semestre.


O objetivo é criar um modelo preditivo capaz de identificar, com base em dados acadêmicos e socioeconômicos iniciais, se um aluno está em risco de abandonar o curso. Isso permite que a instituição tome ações preventivas ou personalizadas.

---

Etapas:

1. Exploração e análise dos dados
2. Limpeza e transformação
3. Engenharia de features com foco em comportamento acadêmico
4. Codificação de variáveis categóricas e padronização
5.Rebalanceamento com SMOTE
6.Treinamento com Random Forest e otimização via RandomizedSearchCV
7. Comparação entre modelo multiclasse original com vazamento e versão binária sem vazamento

---

O impacto: 

- O modelo final consegue prever a evasão de alunos com base apenas no 1º semestre, o que é altamente útil na prática institucional.

- A abordagem usada é próxima da adotada por instituições reais de ensino que aplicam predição de evasão como ferramenta estratégica.
